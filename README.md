## Quick start

1. Create and activate a virtual environment (optional but recommended):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
``` 

2. Install dependencies (if a `requirements.txt` exists):

```powershell
pip install -r requirements.txt
```

3. Run the demo:

```powershell
streamlit run app_main.py
```
